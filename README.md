# nálisis de Ventas E-commerce - Proyecto BI Consulting

## Descripción del Proyecto 

Este proyecto presenta un análisis de datos de ventas para un e-commerce ficticio (al cual llamé "Kiroo") durante el año 2024. Fue desarrollado como parte del proyecto final del curso Business Intelligence en Talento Tech. 

El objetivo principal fue observar y demostrar cómo el Business Intelligence puede utilizar datos crudos en información para la toma de decisiones. El trabajo abarca desde la limpieza y preparación de datos hasta la creación de dashboards interactivos y un storytelling con recomendaciones en toma futura de decisiones. 

## Metodología 
Este proyecto aplica los fundamentos de Business Intelligence vistos durante la cursada, los cuales se organizaron en cuatro etapas:

### ETAPA 2: Preparación y Análisis de Datos (Google Sheets) 

- **Limpieza de Datos:** Normalización de formatos de fecha (MM/DD/AAAA → DD/MM/AAAA), eliminación de filas duplicadas y corrección de valores inconsistentes.
- **Integración de Datos:** Mediante funciones como `BUSCARX`, se agregaron `Precio_Unitario`, `Total_Venta`, `Categoría` y clasificación `Fresco` vs `Procesado` al dataset.
- **Análisis Exploratorio:** Hallazgos clave y cálculo de estadísticas (media, mediana, moda).

### ETAPA 3: Dashboards y Visualización Interactiva (Looker Studio)

- Creación de un dashboard interactivo con filtros por producto, categoría y período, visualizando los KPIs más relevantes.
